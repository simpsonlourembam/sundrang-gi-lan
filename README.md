# Sundrang-gi-lan

A browser-based space shooting game.

## How to Play

**Objective:** Destroy incoming asteroids.

**Controls:**

*   **Computer:**
    *   Move: Use the **Arrow Keys** (Up, Down, Left, Right).
    *   Firing is automatic.
*   **Mobile:**
    *   Move: Touch and drag anywhere on the screen with **one finger** to move the spaceship.
    *   Firing is automatic.

Good luck!
